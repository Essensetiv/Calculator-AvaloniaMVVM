# Калькулятор на Avalonia UI с использованием MVVM Community Toolkit

Это учебное приложение-калькулятор, реализованное с использованием фреймворка [Avalonia UI](https://avaloniaui.net/) и паттерна проектирования **MVVM** с применением [CommunityToolkit.Mvvm](https://learn.microsoft.com/ru-ru/dotnet/communitytoolkit/mvvm/).

## Особенности

- Современный UI с тёмной темой
- Полностью реализован по паттерну MVVM
- Использование атрибутов из CommunityToolkit.Mvvm для упрощения кода:
  - `[ObservableProperty]`
  - `[RelayCommand]`
- Поддержка основных операций: сложение, вычитание, умножение, деление
- Дополнительные функции: смена знака, вычисление процента, очистка

## Функционал

- **Цифры (0-9)** — ввод чисел
- **Операции (+, −, ×, ÷)** — выполнение математических действий
- **=** — получение результата
- **C** — полная очистка
- **+/-** — смена знака числа
- **%** — вычисление процента от числа
- **.** — добавление десятичной точки

## Архитектура проекта

Проект разделён по слоям согласно MVVM:

- `Views/` — XAML-файлы с интерфейсом
  - `MainWindow.axaml` — главное окно калькулятора
- `ViewModels/` — логика представления
  - `MainWindowViewModel.cs` — обработка команд, вычисления, состояние
- `Models/` — модельные классы (при необходимости)
- `ViewModelBase.cs` — базовый класс ViewModel с поддержкой уведомлений

## Технологии

- **Avalonia UI** — кроссплатформенный XAML-фреймворк
- **.NET 6+** — целевая платформа
- **CommunityToolkit.Mvvm** — реализация MVVM с source generators

## Как запустить

1. Убедитесь, что у вас установлен [.NET SDK](https://dotnet.microsoft.com/download)
2. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/yourname/Calculator.git
   cd Calculator
   ```
3. Запустите проект:
   ```bash
   dotnet run
   ```
